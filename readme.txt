3rd Capstone in Zuitt - Coding Bootcamp Philippines; A concert ticketing reservation system with CRUD Operations that used Linux, Apache, MySQL, PHP, HTML, CSS, JavaScript, Bootstrap, Sass, Jquery, Fetch API, Stripe, MongoDB, Express, Node, React, and GraphQL

1. create server folder
2. enter 'npm init' in server folder
3. install packages 'npm install express graphql apollo-server apollo-server-express nodemon mongoose bcrypt body-parser jsonwebtoken'
4. npx create-react-app client in csp3 folder
